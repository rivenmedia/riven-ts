# Settings

## TmdbSettings

_Object containing the following properties:_

| Property          | Description                           | Type                       | Default |
| :---------------- | :------------------------------------ | :------------------------- | :------ |
| `enabled`         | Whether the plugin is enabled or not. | `boolean`                  | `true`  |
| **`apiKey`** (\*) | Your TMDB API Key                     | `string` (_min length: 1_) |         |

_(\*) Required._
