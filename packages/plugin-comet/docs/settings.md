# Settings

## CometSettings

_Object containing the following properties:_

| Property | Description                                  | Type             | Default                       |
| :------- | :------------------------------------------- | :--------------- | :---------------------------- |
| `url`    | The URL of the Comet instance to connect to. | `string` (_url_) | `'https://comet.feels.legal'` |

_All properties are optional._
