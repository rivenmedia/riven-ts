# Settings

## SubdlSettings

_Object containing the following properties:_

| Property          | Description                   | Type                              | Default  |
| :---------------- | :---------------------------- | :-------------------------------- | :------- |
| **`apiKey`** (\*) | Your Subdl API Key            | `string` (_min length: 1_)        |          |
| `languages`       | List of languages to download | `Array<string (_min length: 1_)>` | `["en"]` |

_(\*) Required._
